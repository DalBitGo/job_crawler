## 2. 사업 내용

### 2.1 핵심 사업
**OTT 스트리밍 플랫폼** - 국내 대표 토종 OTT 서비스

### 2.2 주요 서비스/제품
| 서비스명 | 설명 | 현황 |
|---------|------|------|
| 티빙 (OTT) | 영상 스트리밍 서비스 | 핵심 서비스 |
| 티빙 오리지널 | 자체 제작 콘텐츠 | 차별화 전략 |
| 티빙 스포츠 | KBO 리그 등 스포츠 중계 | 2024년 흥행 |
| AVOD | 광고 기반 무료 시청 | 가입자 확대 중 |

### 2.3 서비스 규모
| 지표 | 수치 |
|------|------|
| 유료 가입자 | 약 **800만명+** |
| 목표 가입자 (2027) | **1,500만명** |
| 2024년 매출 | **4,353억원** (+33.4% YoY) |

### 2.4 기술 스택 (추정)

| 분류 | 기술 |
|------|------|
| 스트리밍 | HLS, DASH, DRM |
| 데이터 | Kafka, Spark (실시간 시청 데이터) |
| 추천 | ML/AI 기반 콘텐츠 추천 |
| 클라우드 | AWS (추정) |
| 워크플로우 | Airflow (배치 파이프라인) |

**OTT 특성상 필요한 데이터 처리:**
- 실시간 시청 로그 수집 (Kafka)
- 시청 패턴 분석 (Spark)
- 콘텐츠 추천 시스템
- A/B 테스트 데이터 파이프라인
