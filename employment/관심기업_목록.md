## 검색 팁

### LinkedIn 검색 URL 패턴
```
https://www.linkedin.com/jobs/search/?keywords=Data%20Engineer&location=South%20Korea&f_C={회사ID}
```

### 유용한 키워드
- Data Engineer / 데이터 엔지니어
- Platform Engineer / 플랫폼 엔지니어
- DevOps Engineer
- Backend Developer (데이터 팀)
- MLOps Engineer

### 필터 조합
```
경력: 3-5년 (Mid-Senior level)
게시일: Past week
근무형태: Full-time
위치: Seoul, South Korea
```
